<Warning>
  The resources (image, video, voice) generated by our API are valid for 7 days.
  Please save the relevant resources as soon as possible to prevent expiration.
</Warning>

## Important Notes

- The 'akool_faceswap_image_hq' model only supports high-quality face swapping for a single face in an image
- This simplified API does not require the `opts` parameter, making it easier to use
- Face enhance must be a boolean value (true/false), default is false
