---
title: "Create Character Swap"
description: "Create animated video from character image and source video using advanced animation models (WaveSpeed AI Wan 2.2 and Kling V2.6 Motion Control)"
---

<Warning>
  The resources (image, video) generated by our API are valid for 7 days.
  Please save the relevant resources as soon as possible to prevent expiration.
</Warning>

## Endpoint

```
POST https://openapi.akool.com/api/open/v4/characterSwap/create
```

## Request Headers

| **Parameter** | **Value**    | **Description**                                                                                                                                               |
|---------------|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| x-api-key | API Key | Your API Key used for request authorization. If both Authorization and x-api-key have values, Authorization will be used first and x-api-key will be discarded. |
| Authorization | Bearer `{token}`| Your API Key used for request authorization.[Get Token](/authentication/usage#get-the-token) |

## Body Attributes

| **Parameter** | **Type** | **Required** | **Description**                                                                                                                                        |
|---------------|----------|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| image         | String   | true         | Character image URL (PNG or JPEG) to be animated. For Kling models: max 10MB, min 300px width/height, aspect ratio 1:2.5 to 2.5:1 |
| video         | String   | true         | Source video URL (MP4) with motion to apply. For Kling models: must contain a clearly visible face, 3-30 seconds |
| duration      | Number   | false        | Video duration in seconds (1-120 for WaveSpeed, 3-30 for Kling models). If not provided, will be automatically detected from the video |
| resolution    | String   | false        | Output resolution: `480p`, `720p`, `1080p` (default: `720p`). Only applicable for `wavespeed-ai/wan-2.2/animate` model |
| mode          | String   | false        | Animation mode: `animate` (character animation) or `replace` (face replacement) (default: `animate`). Only applicable for `wavespeed-ai/wan-2.2/animate` model |
| prompt        | String   | false        | Optional prompt text for generation guidance (max 5000 characters for Kling models, max 500 for WaveSpeed) |
| negative_prompt | String | false        | Negative prompt to exclude unwanted elements (max 2000 characters). Only applicable for Kling models |
| model_name    | String   | false        | Model to use for generation. Options: `wavespeed-ai/wan-2.2/animate` (default), `kwaivgi/kling-v2.6-pro/motion-control`, `kwaivgi/kling-v2.6-std/motion-control` |
| character_orientation | String | false  | Character orientation source: `image` or `video` (default: `video`). Only applicable for Kling models |
| keep_original_sound | Boolean | false   | Whether to retain the original video sound (default: `true`). Only applicable for Kling models |
| webhookurl    | String   | false        | Callback URL for POST requests when task is completed |

## Response Attributes

| **Parameter** | **Type** | **Description**                                                                                                                                                                                                        |
| ------------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| code          | Integer  | Interface returns business status code (1000:success)                                                                                                                                                                   |
| msg           | String   | Interface returns status information                                                                                                                                                                                   |
| data          | Object   | Response data object |
| - _id         | String   | Document ID |
| - create_time | Long     | Creation timestamp |
| - uid         | Integer  | User ID |
| - team_id     | String   | Team ID |
| - status      | Integer  | Task status: 1=queueing, 2=processing, 3=completed, 4=failed |
| - video_duration | Number | Video duration in seconds |
| - deduction_credit | Integer | Credits deducted |
| - resolution  | String   | Video resolution |
| - file_name   | String   | Output file name |
| - image_url   | String   | Input image URL |
| - video_url   | String   | Input video URL |
| - prompt      | String   | Animation prompt |
| - mode        | String   | Animation mode |
| - model_name  | String   | Model name used |
| - webhookUrl  | String   | Callback URL |
| - character_orientation | String | Character orientation source (Kling models only) |
| - keep_original_sound | Boolean | Whether original sound is retained (Kling models only) |

## Important Notes

- **Supported Models**: Three models are available - WaveSpeed AI Wan 2.2 (default), Kling V2.6 Pro Motion Control, and Kling V2.6 Std Motion Control
- **Image Quality**: Use high-resolution images with clearly visible faces for better results. For Kling models: max 10MB, min 300px width/height, aspect ratio between 1:2.5 and 2.5:1
- **Video Format**: MP4 format is required for source videos. For Kling models, the video must contain a front-facing person with a clearly visible face
- **Duration**:
  - If not specified, duration will be automatically detected from the source video
  - WaveSpeed model: 1-120 seconds
  - Kling models: 3-30 seconds
- **Mode Options** (WaveSpeed model only):
  - `animate`: Applies character animation to the image using the video's motion
  - `replace`: Replaces faces in the video with the character from the image
- **Prompt**: Use the `prompt` parameter for enhanced control over the animation. Max 5000 characters for Kling models, max 500 for WaveSpeed
- **Negative Prompt** (Kling models only): Use to specify elements you want to exclude from the output (max 2000 characters)
- **Character Orientation** (Kling models only): Set to `image` to follow the character's orientation from the image, or `video` to follow the video's orientation
- **Keep Original Sound** (Kling models only): Set to `true` to retain the original video's audio track
- **Resolution**: Choose between `480p`, `720p`, or `1080p` based on your needs (only applicable for WaveSpeed model). Higher resolution may take longer to process
- **Resource Expiration**: Generated videos are valid for 7 days, save them promptly
- **Webhook**: Use `webhookUrl` to receive notifications when video generation is complete
- Save the `_id` from the response to check video status using the [Get Video Info Result](/ai-tools-suite/character-swap/get-result) API
